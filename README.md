# Expression solver

Made in C++ using object oriented programming :)

## Requirements
- g++ compiler

## How to run the code
1. Clone or download the repository
2. Compile the files by running:
```
g++ main.cpp -o main
```
3. Run the compiled executable file:
```
.\main.exe
```
